# 前提要素 - Part 1

## .NET 6
import { FaDownload } from "react-icons/fa6";

Unreal Build Toolsを正しく機能させるには、システムに.NET 6がインストールされている必要があります。

[ダウンロード <FaDownload />](https://dotnet.microsoft.com/en-us/download/dotnet/6.0/runtime?cid=getdotnetcore&os=windows&arch=x64)

ランタイムをダウンロードするには`Download x64`ボタンをクリックして下さい:

![Dotnetのダウンロード](./assets/Dotnet.png)

インストーラーをダウンロードしたら、それを開いてランタイムをインストールします。

## Visual Studio 2022

SDKをコンパイルするには、Visual Studio 2022がインストールされている必要があります。

[ダウンロード <FaDownload />](https://visualstudio.microsoft.com/vs/)

Community Editionで十分です。インストールする際、以下のコンポーネントを確認して下さい:


![Visual Studioのインストーラーの設定](./assets/VisualStudioInstaller.png)
![Visual Studioのインストーラーの設定2](./assets/VisualStudioInstaller2.png)

確認したらcontinueを押し、Visual Studioがインストールされるのを待ちます。

import { FaCirclePlus } from "react-icons/fa6";
import Highlight from '@site/src/components/Highlight';

## Unreal Engine 5.1

Visual Studioのインストールが終わったら、Unreal Engineをインストールする必要があります。

1) Epic Games Launcherを開く
2) `Unreal Engine`タブを開く
3) `Library`タブを開く
4) <Highlight color="#dea309">Gold <FaCirclePlus /></Highlight>を押して新しいバージョンを追加する

![新しいUnreal Engineのバージョンの追加](./assets/EpicUnrealVersion.png)

5) エンジンのバージョンを選択できる新しいカードが表示される
6) バージョン`5.1`を選択する

![5.1バージョン](./assets/Versions.png)

:::warning
5.1系であれば動作する。5.1でも5.1.1でも問題ない。
:::

## Wwise

Palworldはサウンドに[Wwise](https://www.audiokinetic.com/en/products/wwise/)を使用しているため、これもインストールする必要がある。

:::danger
例えサウンドを作る予定がなくともWwiseは必要です。プロジェクトがコンパイルできなくなってしまいます。
:::

1) [Wwise](https://www.audiokinetic.com/en/products/wwise/)をインストールするために`Download Audikinetic Launcher`をクリック [ダウンロード <FaDownload />](https://www.audiokinetic.com/en/download)


![Audiokinetic Launcherのダウンロード](./assets/WwiseWebsiteDownload2.png)

:::note
ダウンロードのためにアカウントを作る必要があります。
:::

2) ランチャーをダウンロードしたら、インストールしログイン
2) `Wwise`タブを開く
3) 画面の左下に`Install A New Version`ヘッダーがある
4) [Wwise](https://www.audiokinetic.com/en/products/wwise/)のバージョンとして`2021.1.11`を選択しInstallを押す

![新しいバージョンのインストール](./assets/WwiseInstallNewVersion.png)

:::info
次のページでは以下を選択して下さい:

- SDK (C++)
- Microsoft -> Windows -> Visual Studio 2022
:::

選択は以下のようになるはずです:

![インストールの設定](./assets/WwiseInstallSelection.png)

5) `Next`を押し、どのプラグインも必要ないのでそのまま`Install`を押しましょう!
6) [Wwise](https://www.audiokinetic.com/en/products/wwise/) SDKのインストールが終わったら、`Unreal Engine`タブを開く
7) <FaDownload /> ボタンを押して、`Offline Integration Files`を選択する

![オフライン統合ファイル](./assets/WwiseOfflineIntegrationFiles.png)

8) `Integration Version`ではWwiseのバージョンとして`2021.1.11`を選択します。ディレクトリを分かりやすい場所に設定し、インストールを押します!
