# Voraussetzungen - Teil 1

## .NET 6
import { FaDownload } from "react-icons/fa6";

Damit die Unreal Build Tools korrekt funktionieren, muss .NET 6 auf dem System installiert sein.

[Herunterladen <FaDownload />](https://dotnet.microsoft.com/en-us/download/dotnet/6.0/runtime?cid=getdotnetcore&os=windows&arch=x64)

Klicke auf `Download x64` um die Runtime herunterzuladen:

![Dotnet download](./assets/Dotnet.png)

Sobald das Installationsprogramm heruntergeladen ist, öffne es und installiere die Runtime.

## Visual Studio 2022

Um die sdk zu kompilieren, muss Visual Studio 2022 installiert sein.

[Herunterladen <FaDownload />](https://visualstudio.microsoft.com/vs/)

Die community edition reicht aus. Während der Installation müssen die folgenden Komponenten im Installationsprogramm ausgewählt werden:


![Visual Studio Installer Options](./assets/VisualStudioInstaller.png)
![Visual Studio Installer Options 2](./assets/VisualStudioInstaller2.png)

Drücke dann auf "Weiter" und warte auf die Installation von Visual Studio.

## Unreal Engine 5.1

Nach der installation Visual Studio, muss Unreal Engine installiert werden.

1) Öffne den Epic Games Launcher
2) Gehe zum `Unreal Engine` tab
3) Gehe in den `Library` tab
4) Drücke auf <Highlight color="#dea309"><FaCirclePlus /></Highlight> um eine neue Version hinzuzufügen

![Add new Unreal Engine version](./assets/EpicUnrealVersion.png)

5) Nun sollte eine neue Karte mit der möglichkeit die Version auszuwählen sichtbar sein.
6) Wähle Version `5.1`

![5.1 Version](./assets/Versions.png)

:::warning
Jede 5.1 Version funktioniert, es ist egal, ob es 5.1 oder 5.1.1 ist
:::

## Wwise

Palworld nutzt [Wwise](https://www.audiokinetic.com/en/products/wwise/) für Ton, also müssen wir auch dies Installieren.

:::danger
Selbst wenn du nicht vor hast etwas an den Tönen zu ändern, musst du dennoch wwise installieren, ansonsten kannst du das Projekt nicht kompilieren
:::

1) Um [Wwise](https://www.audiokinetic.com/en/products/wwise/) zu installieren, klicke `Audikinetic Launcher herunterladen` [Herunterladen <FaDownload />](https://www.audiokinetic.com/en/download)


![Download Audiokinetic Launcher](./assets/WwiseWebsiteDownload2.png)

:::note
Dies erfordert, das ein Nutzeraccount erstellt wird.
:::

2) Nachdem du den Launcher heruntergeladen hast, installiere ihn und melde dich an
2) Gehe zum `Wwise` tab.
3) Unten links auf deinem Bildschirm sollte nun `Neue Version installieren` sichtbar sein.
4) Wähle `2021.1.11` als deine [Wwise](https://www.audiokinetic.com/en/products/wwise/) Version und klicke auf Installieren

![Install A New Version](./assets/WwiseInstallNewVersion.png)

:::info
Auf der nächsten Seite, wähle:

- SDK (C++)
- Microsoft -> Windows -> Visual Studio 2022
:::

Die Auswahl sollte so aussehen:

![Install Options](./assets/WwiseInstallSelection.png)

5) Nun klick `Weiter`, es werden keine Plugins benötigt, also drücke `Installieren`!
6) Nach der Installation von der [Wwise](https://www.audiokinetic.com/en/products/wwise/) SDK, geh zum `Unreal Engine` Tab.
7) Klicke den <FaDownload /> Knopf, und wähle `Offline Integrations Datein`

![Offline integration files](./assets/WwiseOfflineIntegrationFiles.png)

8) Wähle unter `Integrations Version` die Version `2021.1.11` als Wwise-Version. Legen dann das Verzeichnis an einem denkwürdigen Ort fest, und klicke auf Installieren!
